Обращение за помощью

Прежде чем обращаться за помощью по поводу работы журнала заявок по бриллиантам вы должны ознакомиться с настоящей справочной системой. Если вы обратитесь с вопросом, который описан в настоящей справочной системе, то ваш вопрос будет проигнорирован и оставлен без ответа.

Создание и редактирование заявки

Черновик

Если у вас появилась идея для бриллианта, то вам, в первую очередь, необходимо создать черновик. Каждому черновику присваевается уникальный цифровой номер, который сохраняется на всем пути от черновика до бриллианта.

Ваши черновики доступны только вам. Вы можете их сохранять и редактировать, удалять или отправлять на согласование на право быть бриллиантом.

Отправка черновика на согласование происходит по кнопке "Подать заявку" в списке черновиков. В этом случае ваш черновик станет заявкой на бриллиант.

Заявка на бриллиант

Ваши заявки на бриллианты доступны для просмотра и редактирования вам и членам Совета по блиллиантам. Пока ваша заявки находится на рассмотрении у вас есть возможность отозвать заявку - в этом случае она попадет в список отмененных заявок.

Члены Совета по бриллиантам могут:

  • Одобрить вашу заявку. Заявка попадает в список бриллиантов. С этого момента её могут видеть все сотрудники Компании. Сделать её видимой или невидимой для определенной группы лиц нельзя.
  • Отклонить вашу заявку. Заявка попадет в "Список отмененных заявок". В заявке будет указана причина отказа и автор.

Бриллиант

После одобрения ваша заявка превращается в бриллиант.

Являясь автором бриллианта (другими словами лидером) вы можете:

  • Добавлять членов группы по бриллианту
  • Удалять членов группы по бриллианту
  • Редактивать текст бриллианта
  • Оставлять комментарии по бриллианту

Члены совета по бриллиантам могут:

  • Редактивать текст бриллианта
  • Оставлять комментарии
  • Успешно закрывать бриллиант - с этого момента он считается выполненным

Форматирование текста

Для того, чтобы ваш бриллиант было удобно читать вы можете воспользоваться следующими правилами для редактирования текста.

Заголовки и подзаголовки

Правило
Укажите три или четыре символа решетки # перед текстом. Это придаст тексту вид заголовка
Пример
### Это будет крупный текст
#### Это будет мелкий заголовок

Выделение текста

Правило
Текст курсивом создается через звездочки или символы подчеркивания с обоих концов текста *text* (или _text_). Жирный текст создается через звездочки или двойное подчеркивание с обоих концов текста *text* (или __text__).
Пример
*Этот текст будет курсивом* → Этот текст будет курсивом
_Этот текст также будет курсивом_ → Этот текст также будет курсивом
**Этот текст будет жирным** → Этот текст будет жирным
__Этот текст также будет жирным__ → Этот текст также будет жирным
*Но вы **можете** комбинировать стили* → Но вы можете комбинировать стили

Создание ненумерованных списков

Правило
Создание ненумерованных списоков осуществляется через одинарную звездоку и пробел после звездочки, но перед текстом * text
Пример
* Позиция 1
* Позиция 2
Результат
  • Позиция 1
  • Позиция 2

Создание нумерованных списков

Правило
Создание нумерованных списков осуществляется через указание арабских цифр с последующей точкой 1. text
Пример
1. Позиция 1
2. Позиция 2
Результат
  1. Позиция 1
  2. Позиция 2

Прикрепление ссылки

Правило
Создание ссылки на другой ресурс: в квадратных скобках указываете название, в круглых - ссылку. [тут пишете название ресурса как вам угодно](тут url)
Пример
Cсылка на файловое хранилище: [Это ссылка на файл](http://filerepo.a-i.kz/login) будет выглядеть так: Это ссылка на файл

Номер заявки

Номер заявки состоит из арабских цифр. Этот номер служит для идентификации вашей заявки на пути "черновик" → "заявка" → "бриллиант". Номер заявки не обозначает количество заявок в системе.

Совет по бриллиантам

Все заявки на бриллианты рассматривает Совет по бриллиантам. Члены совета видят все заявки на бриллианты. Посмотреть список тех, кто входит в совет по бриллиантам вы можете пройдя по ссылке "Совет по бриллиантам" на главной странице.

Участие в работе группы по блиллиантам

Автор (только автор) бриллианта может добавлять вас в группу по работе над бриллиантом. Удалить из группы также может только автор бриллианта. Если вы являетесь членом группы, то вы можете оставлять комментарии.

На главной странице вы можете увидеть список бриллиантов, в которых вы участвуете. В этой же строке указано количество бриллиантов, в которых вы участвуете.
